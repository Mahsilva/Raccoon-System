# Raccoon-System

# Client & Booking Management System

## 📌 Project
This project is a management system for handling customers and service bookings.  
It is currently under development and being built as part of my software development portfolio.

## 🚀 Features (In Progress)
- Customer registration
- Service booking
- Booking schedule management
- Service price tracking
- Notes for bookings
- Booking time control (start time and end time)

## 🛠️ Technologies
- C#
- .NET
- ASP.NET MVC
- Entity Framework
- SQL Server

## 📊 Status
🚧 Work in progress

New features and improvements are being added regularly.

## 🎯 Goal
The goal of this project is to practice backend development, database integration, and system architecture while building a real-world management system.
